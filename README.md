# 🌍 EaseStay - Your One-Stop Travel Booking Solution

**EaseStay** is a modern, responsive, and user-friendly travel booking web application designed to simplify and enhance the travel planning experience. With options to book **hotels, flights, and car rentals**, EaseStay brings convenience and efficiency into one unified platform.

## 🚀 Live Demo

🔗 [Visit EaseStay Website]([https://divyamoswal.github.io/EaseStay/](https://ease-stay-eta.vercel.app/index.html))  
*(Live link will work if deployed via GitHub Pages)*

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Responsive Web Design (RWD)**
- **VS Code** for development

## ✨ Features

- 🏨 Hotel booking section
- ✈️ Flight booking with search and top deals
- 🚗 Car rental services
- 📱 Responsive design for all devices
- 👤 Login/Signup (Google, Facebook, Apple Sign-in integrated)
- 📧 Newsletter subscription
- 💬 Testimonials & contact form
- ⚡ Smooth UI animations and interactive elements
